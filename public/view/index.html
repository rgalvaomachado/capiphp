Bem-vindo ao CapiPHP

O CapiPHP é um framework PHP leve e eficiente, projetado para simplificar o desenvolvimento de aplicações web robustas e escaláveis. Ele oferece uma arquitetura modular, facilitando a organização do código e a integração de novas funcionalidades.

Configuração do Ambiente

Siga os passos abaixo para configurar o ambiente corretamente:

Pré-requisitos:

    1. Banco de dados disponível para importação.
    2. Servidor com suporte a PHP.

Passos para Configuração:

  1. Importar o banco de dados: O banco está localizado em /database.
    Importe-o para o seu sistema.
  2. Configurar env.php: Edite o arquivo env.php com as informações do banco de dados e infraestrutura:

Após configurar tudo, você pode testar a conexão acessando a aplicação no navegador.
